# 💻 [2026-01] 나만의 C언어 소프트웨어 개발 프로젝트

## 1. 시나리오 제목
* 우주 먼지 토끼 키우기(다마고치)

<br>


## 2. 시나리오 (5~10줄)
*이 프로그램은 광활한 우주를 떠다니는 신비로운 존재인 '우주 먼지 토끼'를 수집하고 성장시키는 육성 시뮬레이션 시스템입니다. 사용자는 토끼의 출신 은하, 초기 질량, 수집한 별 가루 개수 등 기초 데이터를 입력하여 관리의 시작을 알립니다. 시스템은 입력된 데이터를 바탕으로 토끼의 현재 상태와 반짝임 지수를 계산하며, 향후 업데이트를 통해 질량에 따른 진화 등급(소행성급, 행성급, 블랙홀급 등)을 판정하고 특별한 우주 아이템을 지급하는 로직을 포함할 예정입니다. 최종적으로는 무한 루프 시스템을 통해 먹이 주기, 여행하기 등의 메뉴를 자유롭게 선택하고, 여러 마리의 먼지 토끼(배열)를 동시에 관리하며 은하계 최고의 반려 토끼로 키워내는 종합 관리 소프트웨어를 완성하는 것이 목표입니다.

<br>


## 3. 예상 기능 및 메뉴 (최소 5개)

1. 신규 우주 먼지 토끼 등록: 우주 먼지 토끼의 기초 데이터 설정 -변수 선언
2. 성장 상태 분석: 토끼의 상세 스탯 계산 - 산술 연산
3. 토끼 관리 및 상호작용: 사용자의 선택에 따른 스탯 변화 - if-else, switch
4. 진화 등급 판정: 스탯 값에 따른 진화 단계 산출(SSS~F) - if-else
5. 업적 및 칭호: 일정 목표치 달성 시 업적 달성 출력 및 칭호 부여 - if-else
6. 우주 탐사 미니게임: 별가루 채집 이벤트 및 텍스트 연출 - while
7. 성단 관리 시스템: 보관 중인 여러 토끼들의 정보값 일괄 출력 - array
0. 시스템 종료: 프로그램 종료

## 🟦 [1차 과제: V1.0] 기초 뼈대 구축 (입출력과 변수)
### **✨V1.0 개발 내용:**
  * 사용자로부터 토끼의 이니셜(문자), 초기 질량(실수), 수집한 별가루 개수(정수)를 입력받아 최종 질량과 반짝임 지수를 계산해 출력하는 기능을 구현했습니다.
  * 산술 연산을 통해 별가루 섭취량에 따른 질량 증가값과 반짝임 지수를 자동으로 산출하는 로직을 적용했습니다.
    
### **🤖 AI 파트너십 과정**
 1. **내용 1**
    * **프롬프트 요약:** "C언어 4단계 과제 로드맵(입출력 ➔ 조건문 ➔ 반복문 ➔ 배열/함수)에 맞도록 우주 먼지 토끼 키우기의 단계별 기능 확장 로드맵 요청"
    *  적합한 RPG 캐릭터 성장 시나리오 및 단계별 기능 확장 로드맵 기획 요청"
    * **적용 내용:** 단순한 스탯 계산기를 넘어, '스탯 입력(1차) ➔ 전투력 등급 판별(2차) ➔ 메인 메뉴 시스템(3차) ➔ 다중 캐릭터 관리(4차)'로 이어지는 논리적인 시스템 확장 로드맵과 예상 메뉴를 확립함.
      
 2. **내용 2**
    * **프롬프트 요약:** "1차 과제 필수 요건(변수 5개, 자료형 3개 이상)을 충족하기 위한 캐릭터 초기 스탯 데이터 구조 모델링 논의"
    * **적용 내용:** 포괄적인 의미의 '공격력' 변수 하나를 사용하는 대신, 데이터 특성에 맞춰 문자형(`char` - 이니셜), 정수형(`int` - 체력, 마나), 실수형(`float` - 무기 공격력)으로 자료형을 명확히 분리하여 입력 데이터의 구조적 완성도를 높임.
    
### **🛠️ Troubleshooting & 기술 회고:**
  1. **문제 1:** `scanf`로 문자를 입력받을 때 버퍼 문제로 엔터키가 입력되는 현상이 발생함.
     * **해결:** AI에게 질문하여 `scanf(" %c", &initial);` 처럼 `%c` 앞에 공백을 두어 버퍼를 비우는 팁을 얻고 코드를 수정하여 해결함.
  2. **문제 2:** 실수형 데이터(무기 공격력)를 출력할 때 소수점이 너무 길게 나옴.
     * **해결:** 구글링과 AI 검색을 통해 `printf("%.2f", weapon_power);`와 같이 서식 지정자를 수정하여 소수점 둘째 자리까지만 출력되도록 개선함.
